## Templates

### Template A — Premier contact (défaut)

**Quand :** post récent fort, première prise de contact.

**Infos requises :**
- @handle ou nom du compte
- Détail du post à citer (marque, concept, sujet)
- Compliment spécifique (lumière, composition, idée…)

```
Hi [Name] 👋

Really liked your [POST_DETAIL] — [SPECIFIC_COMPLIMENT].

Quick question: do you ever batch product visuals for clients, or is it mostly one-off creative posts?

We built Image Studio inside ViralWorks — 2K product visuals with NanoBanana Pro + GPT 2.0, all in one place. We're offering a 7-day free trial with 30 images, no commitment.

No pressure — here's the link if you want to test it on a real brief:
https://viralworks-studio.com/image-studio

Have a great week!
```

## Règles de sélection

| Situation | Template |
|-----------|----------|
| Premier contact, post identifiable | **A** |
| Gros compte / très sollicité | **B** |
| Agence output client régulier | **C** |
| Relance après 3 j sans réponse | **D** |
| Réponse positive reçue | **E** |
| Post flou / infos insuffisantes | **Demander infos avant rédaction** |

---

## Ne jamais

- Dire « link in my bio » — toujours le lien direct Image Studio
- Inventer un post ou un compliment non visible dans les infos fournies
- Dépasser ~120 mots en premier DM
- Relancer plus d'une fois
- Ne jamais mentionner carte bancaire ou Stripe dans les DMs — dire "sans engagement" / "no commitment"

---

## Objectifs réalistes (cold DM)

| Métrique | Cible |
|----------|-------|
| Taux de réponse | 10–20 % |
| Taux de clic | 3–8 % |
| Taux d'inscription | 1–3 % |

Sur 20 DMs bien ciblés / semaine → 0 à 1 inscription / semaine au début = normal.

---

## Tunnel recommandé

```
Ciblage (post produit/brand < 7 j)
  → DM personnalisé (Template A ou C)
    → Lien direct image-studio
      → Compte + essai démarré
        → 1ère image générée < 10 min
          → Relance J+3 si besoin (Template D)
```
